This free online tool converts common length, temperature, area, volume, weight, and time units.

[https://www.calculator.io/unit-converter/](https://www.calculator.io/unit-converter/)

== Usage ==

Add the Unit Converter shortcode to your page, post or sidebar:

`[ci_unit_converter]`

Add the following code to your template where you would like the Unit Converter to appear:

`<?php display_ci_unit_converter(); ?>`

== Installation ==

1. Upload the Unit Converter /ci_unit_converter/ folder to the /wp-content/plugins/ directory.
2. Activate the Unit Converter plugin through the "Plugins" menu in WordPress.
